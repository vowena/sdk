# Code of Conduct

## Our commitment

We want Vowena SDK to be welcoming and constructive for contributors at every experience level.

## Expected behavior

- Be respectful and specific in feedback.
- Explain tradeoffs when discussing API design.
- Help integrators and newcomers understand the reasoning behind decisions.
- Keep discussions focused on code, behavior, and user impact.

## Unacceptable behavior

- Harassment, discrimination, or hate speech
- Personal attacks or intimidation
- Sharing private information without consent
- Persistent bad-faith disruption

## Scope

This applies in issues, pull requests, code review, discussions, and related project spaces.

## Reporting

Report conduct concerns privately through GitHub when possible. If that is not possible, email `security@vowena.xyz` with the subject prefix `[conduct]`.

## Enforcement

Maintainers may remove content or restrict participation when necessary to keep the project safe and collaborative.
