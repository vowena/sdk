# Support

## Use issues for

- Reproducible SDK bugs
- Scoped feature requests
- Documentation gaps tied to this package

## Before opening an issue

- Read [README.md](README.md) for install and API context.
- Read [CONTRIBUTING.md](CONTRIBUTING.md) for the verification bar.
- Read [SECURITY.md](SECURITY.md) if the issue could be security-sensitive.

## Questions and design discussion

If discussions are enabled for this repository, use them for open-ended integration questions. Otherwise, open an issue with enough context for maintainers to guide you.
